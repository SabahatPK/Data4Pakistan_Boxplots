How does the number of people in Unpaid Employment vary across provinces and gender?
Key Takeaways
  • Excluding ICT, because it is only one data point, each province shows much greater dispersion of data for women than men.
  • The median for women in Unpaid Employment is higher than for men, in each province.
  • There are some 0 values for women in Balochistan and KPK which is a little suspicious.

About This Dashboard:

This dashboard was built using d3.js, a Javascript data visualization library. The repo can be found here. I know there are easier ways to create a boxplot than to code it in d3; but part of my goal is to learn more about d3.
